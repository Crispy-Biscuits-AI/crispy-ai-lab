# Archive

This directory holds experimental, legacy, or no-longer-primary repo material that was moved out of the main product path during cleanup pass two.

Nothing here is part of the current minimum CrispyBrain deployment story.

Archive is for:

- old compose experiments
- repetitive test artifacts
- materials worth preserving for reference but not for default deployment

Archive is not:

- the source of truth for the current product
- a place to pull defaults from during normal setup
